# 📊 Customer Behavior Dashboard Analysis

## 🧠 Overview

This project is an interactive data visualization dashboard built using **Streamlit** to analyze and explore customer behavior patterns.
It transforms raw transactional data into meaningful insights that support data-driven decision making.

---

## 🚀 Key Features

* 📈 Interactive data visualization
* 🧑‍🤝‍🧑 Customer segmentation insights
* 💰 Revenue and purchasing pattern analysis
* 🎯 Dynamic filtering for deeper exploration
* ⚡ Fast and user-friendly dashboard interface

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **Pandas**
* **Matplotlib / Seaborn**

## ▶️ How to Run Locally

1. Clone this repository:

```bash
git clone https://github.com/rapurby/Customer-Behavior-Dashboard-Analysis.git
cd Customer-Behavior-Dashboard-Analysis
```

2. Create virtual environment:

```bash
python -m venv venv
venv\Scripts\activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
